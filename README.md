# 🎯 Jeopardy 2.0

A modern recreation of the classic **Jeopardy!** trivia game built with **HTML, CSS, JavaScript, Fetch API, and jQuery**. This project dynamically loads trivia categories from an external API, supports multiple players, includes a Daily Double feature with a wagering system, and allows games to be restarted with a customizable player roster.

## 🎮 How to Play

1. Enter player names.
2. Add or remove players if needed.
3. Click **Start Game**.
4. Select a clue from the board.
5. Click once to reveal the question.
6. Click again to reveal the answer.
7. Award or deduct points using the scoreboard controls.
8. If a **Daily Double** appears, enter a wager before revealing the clue.
9. Continue until all clues have been played.
10. The winner is announced automatically.
11. Click **Restart Game** to return to the player setup screen.
